pyYaUsbIR
=========

an userspace driver for yaUsbIR written in python

requires:
---------
 * python3
 * python-bitarray
 * python-dbus
 * python-gi
 * pyhton-uinput: https://github.com/tuomasjjrasanen/python-uinput
 * pyusb: https://github.com/walac/pyusb
